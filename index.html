# 介绍

  • MarketUI是推啊创新·活动前端团队开发的一套移动端互动营销组件库

# 特性

  1. 1、提供业务常用的*大互动营销类组件
  2. 2、完善的文档和使用示例
  3. 3、支持按需引入
  4. 4、支持Typescript
  5. 5、更贴近业务

# 浏览器支持

  • MarketUI支持现代浏览器以及Android >= 4.0、iOS >= 8.0
