# 变体与预设

主要是为了更贴近业务，方便业务使用

# 变体

由最基本的玩法组件扩展而来，可能是直接使用最基本的组件，额外增加元素，如基座大转盘；也可能是使用核心的处理逻辑，如方形转盘

# 预设

某个玩法组件（包含变体）有多套预设的配置，方便使用，玩法组件本身就是一套预设，比如大转盘有黄色系的，也有蓝色系的
