# `@tuia/market-ui`

> 丰富、可靠的移动端营销互动组件库

## 安装
```bash
yarn add @tuia/market-ui
```

## 引入

### 方法一、全量导入所有组件

```js
import Vue from 'vue'
import MarketUI from '@tuia/market-ui'
import '@tuia/market-ui/index.less'

Vue.use(MarketUI)
```

### 方法二、手动按需引入组件

```js
import TurnCard from '@tuia/market-ui/lib/TurnCard'
import '@tuia/market-ui/lib/TurnCard/style'
```
